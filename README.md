# 📌 О программе

**Baccarat Algorithm** — это программа, которая анализирует результаты игры **Insurance Baccarat** в реальном времени. Находит определённые патерны в результатах игр, и предлагает свою ставку на основе вычислений на нескольких столах. 

Программа автоматически:

* подключается к игровым столам
* получает результаты раундов
* анализирует последовательности игр
* определяет игровые паттерны
* формирует **рекомендации для следующих ставок**

Алгоритм анализирует структуру **Big Road** и определяет различные игровые ситуации:

* красивые серии
* не красивые серии
* качели
* пропуск хода

На основе анализа программа отображает:

* рекомендуемую ставку
* текущую статистику алгоритма
* количество побед и поражений алгоритма
* текущую серию поражений алгоритма

# 🚀 Первый запуск

При первом запуске программе необходимо **авторизоваться на сайте казино**.

Это требуется для получения доступа к игровому столу.

---

## 🔐 Шаг 1. Запуск программы

Запускайте exe файл:

```
BaccaratAlgorithm.exe
```

После запуска откроется окно программы.

---

## 🌐 Шаг 2. Открытие сайта

Нажмите на кнопку:

```
Открыть зеркало
```

Программа автоматически откроет браузер и перейдёт на актуальное зеркало.

---

## 👤 Шаг 3. Вход в аккаунт

В открывшемся браузере:

1. **дождитесь полной загрузки сайта** _(Важно!)_
2. войдите в свой аккаунт

После этого вернитесь в приложение.

_Иногда при первой попытке запуска программы сайт Riobet может долго загружаться, в этом случае **обязательно нужно дождаться полной загрузки сайта** и только после этого совершить вход и нажать продолжить в программе!_

---

## 💾 Шаг 4. Сохранение сессии

В окне программы нажмите на кнопку:

```
Продолжить
```

Программа сохранит данные авторизации и создаст файл:

```
authentications/auth.json
```

Этот файл позволяет **сохранить данные о вашей сессии**, таким образом вам **не нужно будет каждый раз заходить в свой аккаунт**.

---

## 🎰 Шаг 5. Добавление нового стола

Для того чтобы добавить новый стол для анализа просто нажмите на кнопку добавления стола на сайте, **программа сама заметит новый стол и начнёт его анализировать** и **добавит его в интерфейс**

В интерфейсе появится статистика для нового стола, которая будет содержать те же элементы статистики алгоритма, что и основной стол.

# ⚠️ Возможные предупреждения

## Недостаточно данных

Если последовательность слишком короткая:

```
Алгоритм начнёт работу с восьмой завершённой серии
```

Если последовательность слишком большая:

```
Алгоритм не может продолжать анализ после сороковой серии
```

---

## Ничья

Если алгоритм встретил ничью в предыдущей строке:

```
Алгоритм пропускает текущую серию
```

---

## Критическая серия поражений

Если серия проигрышей превышает допустимое значение, программа отображает предупреждение.
